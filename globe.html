In late 2017 I was contracted to build a web-based 3D interactive visualization for VerticalOps showing the business they managed around the globe in real time.

I built a similar globe visualzation while at InsideSales.com. That project was not browser-based and was built using Processing. VerticalOps wanted something similar, but that could run on the web. So even though the look and feel of the visualization is similar, building this project was very different than the first globe I made and required a different tool set.

3D elements were built using Three.js. 2D charts and graphs built with D3.js and p5.js. THe logo animation was built using After Effects and rendered using Lottie. 3D textures were designed and created using Photoshop and Illustrator.
